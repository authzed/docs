## Getting Started

Install pnpm:

```sh
brew install pnpm
```

Install dependencies:

```sh
pnpm install
```

Run a development server:

```sh
pnpm run dev
```

Now you should be able to see the docs rendered at http://localhost:3000


Run linters:

```sh
pnpm run lint
````
